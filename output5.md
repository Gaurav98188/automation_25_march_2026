• MirandaEngineer のバックアップ機能を用いてバックアップを実施した際の、バックアップ先のフォル

ルダ構成とする。

## (5). 蓄積ID管理ファイルの編集

• バックアップ処理時、蓄積 ID 管理ファイルを以下のとおり編集する。

-[HISTORY_TOTAL_ID]セクションのID のみの記載とする。

\- ID には、バックアップした蓄積ファイルのうち、最新の蓄積ファイルの行を示す IDを記載する。


## (6). 蓄積データリストファイルの編集

•バックアップ処理時、蓄積データリストファイルを以下のとおり編集する。

\- 転送する蓄積 ID管理ファイル内のセクション:HISTORY_TOTAL_ID、キー:ID の値を取得する。

\- 下記仕様で蓄積データリストファイルを編集する。

1行目
:[HISTORY_INFO] 固定

2行目以降:1IDの行まで無効行(下図の先頭文字#の行の様式)を書き込む

2バックアップする蓄積ファイルの蓄積 ID • 蓄積サブ ID の行に、バックアップする
該当行の内容を書き込む

※IDの値が10000以上の場合、複数の蓄積データリストファイルを作成する。


図 3-1 蓄積データリストファイルの例(ID=7、蓄積ファイル1ファイルバックアップ時)

## (7). アラーム • イベントデータのバックアップ

イベントデータの全てのファイルをコピーする。

表示設定ファイルをコピーする。
